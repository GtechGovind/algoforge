# SEO Brief and Content Strategy

## Primary audience

- software engineers preparing for coding interviews
- students building algorithm literacy
- teams creating technical hiring practice curriculums

## Long-tail targets we optimize for

- `LeetCode algorithm patterns`
- `data structures for coding interviews`
- `how to study dynamic programming`
- `systematic algorithm study plan`

## On-page strategy

- Use clear pattern-level taxonomy
- Keep notes short, skimmable, and checklist-driven
- Include code in three popular languages
- Link every topic to an implementation file and revision checkpoint
