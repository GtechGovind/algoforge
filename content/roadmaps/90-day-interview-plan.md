# 90-Day Interview Plan

A practical sequence for coding interviews.

## Phase 1: Foundation (Days 1-30)

- Arrays, linked lists, hash map operations
- Two pointers, binary search, recursion fundamentals
- Easy-medium daily coding drills

## Phase 2: Pattern Depth (Days 31-60)

- Dynamic programming (state design + compression)
- DFS/BFS in trees/graphs
- Backtracking and interval/heap questions
- One mock interview every 3 days

## Phase 3: Interview Compression (Days 61-90)

- Timed contests: 2 hr blocks
- Hard problem breakdowns by pattern
- Whiteboard explanation + complexity defense
- Daily postmortem review

## Deliverables

- one topic note per day
- one code summary per week
- two polished whiteboard explanations per week
