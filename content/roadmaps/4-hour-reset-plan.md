# 4-Hour Daily Reset Plan

Use this plan when you need fast retention recovery and confidence rebuilding.

## 4-hour block structure

- 30 minutes: read one core concept
- 60 minutes: solve 1 medium problem without looking up a solution
- 20 minutes: map the solution to a known pattern
- 45 minutes: rewrite same problem from memory
- 25 minutes: implement in a second language
- 20 minutes: flash-review previously missed mistakes
- 10 minutes: plan tomorrow’s concept
