# 00. Overview

## Core idea

AlgoForge is organized around three questions:

- **What problem are we solving?**
- **Which pattern applies?**
- **How do we generalize it?**

## 14-day jumpstart

1. Read one topic per day from `docs/01-foundation`.
2. Solve 2 problems using the topic’s core template.
3. Log mistakes in a daily notebook.
4. Review previous 2 days once every 5 days.

## Learning levels

- `Beginner`: complexity basics + arrays + strings + linked lists
- `Intermediate`: recursion trees, binary search, DFS/BFS, hash, interval + heap topics
- `Advanced`: dynamic programming, graph/advanced data structures, mixed hard problems
