# Interview Arena: 8-Week Loop

## 8-week visual plan

This is a beginner-safe weekly loop:

```
Week 1-2: Foundation
Week 3-4: Core patterns
Week 5-6: Mixed drills
Week 7-8: Mock interviews + reflection
```

## Weekly structure

- **Mon-Wed**: new topic + 2 practice problems
- **Thu-Fri**: mixed drills + previous mistakes rewrite
- **Sat**: mock interview (90 min)
- **Sun**: review + flash summary

## Beginner-friendly target each week

- Week 1: 1 pattern explained simply + 1 tiny implementation
- Week 2: add a second pattern and compare both
- Week 3-4: solve 2 problems/day with same template
- Week 5-6: explain answers to a friend in plain words
- Week 7-8: simulate real interview timing

## 1-hour daily routine

- 10 min: choose 1 pattern
- 30 min: read one canonical problem + variant
- 15 min: code from scratch (no copy-paste)
- 5 min: write one-line lesson learnt

## Milestone rules

- Week 2: explain every solution out loud to a peer or recorder
- Week 4: solve one hard problem per week
- Week 6: do 3 consecutive 90-min mocks
