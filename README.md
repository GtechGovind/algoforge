# AlgoForge Academy

A practical, modern algorithm learning platform built around **patterns**, **clarity**, and **interview confidence**.

## SEO keywords
`algorithms`, `LeetCode`, `data structures`, `coding interview`, `dynamic programming`, `two pointers`, `binary search`, `sliding window`, `systematic coding practice`, `algo roadmap`

## What is this project?

`AlgoForge Academy` is a refreshed and improved version inspired by classical Chinese algorithm notes. It keeps the strongest content while adding:

- English-first roadmap and curriculum design
- Pattern-based learning tracks for interview prep
- Cross-language code templates (Python / Java / JavaScript)
- Weekly study rituals and progress tracking

## Why “AlgoForge”?

The name is built for memorability and discoverability:

- **Algo** → algorithm
- **Forge** → craftsmanship and structured practice

This makes it suitable for SEO while sounding modern and brandable.

## Project structure

- `docs/00-overview`, `docs/01-foundation`, `docs/02-patterns`, `docs/03-interview-arena`, `docs/cheatsheet`: modern study tracks
- `solutions/`: short problem-oriented notes
- `code/`: reusable language implementations
- `content/`: generated indexes and metadata used by curriculum builders
- `scripts/`: project automation scripts

## Why this was rebuilt as a learning-first course

- Visual-first explanations: each topic now follows a simple **see → do → explain** rhythm.
- Beginner-safe pacing: start with one small idea, repeat it, then reuse it in 2-3 problems.
- Interview-ready progression: foundations → patterns → timed practice.
- Cross-language clarity: every idea maps cleanly into Python, Java, and JavaScript examples.

## Learning path for total beginners

Treat every section as a game level:

1. Read the page with the "picture" notes.
2. Run through the `solutions/` problem notes.
3. Implement the same idea in your preferred language from `code/`.
4. Explain it back in one sentence in plain language.

If this feels hard at first, that is expected.
Use the 15-minute study loop from [docs/00-overview](/docs/00-overview/README.md) and finish with one tiny review line:  
`What did I learn today?` `What confused me?`

## Quick start for learners

1. Read [Start Here](docs/00-overview/README.md)
2. Open [Course Guide for Visual Learners](docs/00-overview/visual-learning-guide.md)
3. Choose one category (for example, `docs/01-foundation`)
4. Solve at least two problems and log the pattern used
5. Review the same topics weekly using `docs/03-interview-arena/roadmap.md`

### 15-minute starter flow (perfect for your first day)

- 5 min: read the concept in plain words.
- 5 min: trace the pattern with paper drawings.
- 5 min: code a tiny version without looking at the full solution.
